
# Expense Tracker with AI Assistant 

An Expense Tracker with AI Assistant developed to help users efficiently record, manage, and analyze their daily expenses. The system provides a user-friendly dashboard where users can add expenses, view summaries, analyze spending patterns through charts, and interact with an AI assistant for quick financial insights.




## Used By

Students – to track and manage their daily expenses.

Individuals – to monitor spending habits and maintain financial discipline.

Small Budget Planners – to analyze weekly and monthly expenses through charts.

Personal Finance Users – to quickly check spending insights using the AI assistant.


## Tech Stack

Frontend : HTML, CSS, JavaScript

Backend : Python using Flask

Data Visualization : Chart.js

Data Storage : JSON File

Server : Flask Development Server
